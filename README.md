# Cybersecurity Assignment

This repository contains a cybersecurity assignment covering essential concepts and practical applications in information security. It includes:

* **Research on 5 Real-World Cyberattacks** and their impact on the **CIA Triad** (Confidentiality, Integrity, and Availability).
* **Analysis of 3 Cybersecurity Tools**, including their features, purposes, and real-world use cases.
* **Secure Home Network Design** with a network diagram and security best practices.
* **Essay:** *What Does a Cybersecurity Analyst Do Every Day?* outlining the daily responsibilities, required skills, tools, and career opportunities in cybersecurity.

This project demonstrates an understanding of cybersecurity fundamentals, threat analysis, risk management, and secure network design. It was completed as part of a cybersecurity coursework assignment.
